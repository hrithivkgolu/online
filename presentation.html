NPV Calculation Example (Python)


def npv(cashflows, discount_rate):
    """Compute net present value."""
    return sum(cf / (1 + discount_rate) ** n for n, cf in enumerate(cashflows))

# Example usage
cashflows = [-100000, 30000, 40000, 50000, 45000]
print("NPV @ 8%:", npv(cashflows, 0.08))
# Output: NPV @ 8%: 18,381.61

Key Financial Formulas

Present Value of a single cash flow:

$$ PV = \frac{CF}{(1 + r)^n} $$

Net Present Value (stream):

$$ NPV = \sum_{t=0}^{T} \frac{CF_t}{(1 + r)^t} $$
Tie discount rate (r) to the firm’s WACC and scenario planning.
